## Dependencies
* [Xcode 10](https://developer.apple.com/xcode)
* [Realm](https://realm.io)
* [Ambience](https://github.com/tmergulhao/Ambience)

## Setup
* Clone: `git clone https://github.com/rodrigopasc/Studify`
* Run: `$ pod install`
* Finally: `$ open Studify.xcworkspace`
